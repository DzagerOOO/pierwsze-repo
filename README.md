# Pierwsze repo – hello-world

To prosta strona HTML/CSS przygotowana jako ćwiczenie z systemem kontroli wersji Git oraz serwisem GitHub.

## Jak uruchomić
1. Sklonuj repozytorium: `git clone https://github.com/Dzager000/pierwsze-repo.git`
2. Wejdź do folderu projektu.
3. Otwórz plik `index.html` w dowolnej przeglądarce internetowej.

## Technologie
- **HTML5**: Struktura strony.
- **CSS3**: Stylizacja i układ.
- **Git & GitHub**: Zarządzanie wersjami kodu.
